Privacy Policy

Last updated: May 29, 2026

1. Introduction

BallIdentifier ("we", "us", "our") is committed to protecting your privacy. This Privacy Policy explains what data we collect, how we use it, and what happens when you use the bot, dashboard, website, and related services.

2. Data We Collect

User Data: When you use BallIdentifier, we may collect and store the following information:

Data Type Description
Discord User ID Your unique Discord identifier for authentication and tracking
Username Your Discord username for logging and support purposes
Identification Count Number of successful identifications attributed to your account
Notification Preferences Your optional settings for thank-you messages and upvote reminder DMs
Upvote Cache A local cache of current upvoters fetched from Discord Bot List
Operational Logs Command names, usernames, guild names, and error details needed to operate and debug the service

Server/Guild Data: When the bot is used in a Discord server, we collect:

Data Type Description
Server ID Your Discord server's unique identifier
Notifier Configuration Selected bot IDs, role ID, custom message, setup user, and setup timestamp stored for the dashboard-managed notifier
Guild Usage Counts Aggregated identification totals used for public bot statistics
Note: We do not permanently store raw spawn images. Attachments are processed temporarily to generate hashes, and the temporary file is deleted after processing.
3. Media Processing & Reports

When you use the identify command or notifier features, the bot downloads the attached spawn image, creates a temporary resized copy, computes an image hash, and then removes the temporary file.

If the match confidence is low or an error occurs, the bot may send a report to a configured webhook. Those reports can include your Discord tag or ID, the detected match, the original image URL, the bot involved, and error details.

4. Data Retention

User Data:

  • Your user statistics and notification preferences are stored until you ask us to remove them or stop using the service.

Server Data:

  • Notifier configuration is retained until it is removed through the dashboard or deleted from storage.
  • Aggregated usage counters are kept for service statistics.
5. Where Data is Stored

Data is stored in the bot's asset files on our hosting infrastructure and in any configured third-party services used for bot operations, such as Discord Bot List and optional webhooks.

6. Data Sharing

We do not sell your personal data. We may share limited data only when necessary to run the service, including the following:

  • Discord Bot List for fetching upvote status and posting bot stats
  • Discord DMs for optional thanks messages and upvote reminders
  • Configured log, error, report or upvote webhooks
  • Hosting and storage infrastructure used to run the bot and dashboard
7. Your Rights & Data Deletion
